# Streakly

A private Telegram habit-tracking bot that sends local-time reminders, enables one-tap completion, and tracks streaks with gentle weekly summaries and milestone nudges.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
